# Datos Sismológicos 🌍🌋

Este proyecto está dedicado a la recolección, procesamiento y visualización de datos sismológicos. El objetivo principal es facilitar el análisis de eventos sísmicos mediante herramientas de ciencia de datos, permitiendo identificar patrones, magnitudes y ubicaciones geográficas de manera eficiente.

## 🚀 Características

  - **Procesamiento de Datos:** Limpieza y normalización de catálogos sísmicos.
  - **Visualización:** Generación de gráficos de dispersión, mapas de calor y series temporales de actividad sísmica.
  - **Análisis Estadístico:** Cálculo de estadísticas descriptivas y distribuciones de magnitud (Ley de Gutenberg-Richter).

## 🛠️ Instalación

Para ejecutar el código de este repositorio, asegúrate de tener instalado Python 3.x y las dependencias necesarias.

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/EstebanGonzabay/DatosSismologicos.git
    ```
2.  Accede al directorio:
    ```bash
    cd DatosSismologicos
    ```
3.  Instala las bibliotecas requeridas (ejemplo):
    ```bash
    pip install pandas matplotlib numpy seaborn folium
    ```

## 📊 Uso

Puedes explorar los datos utilizando los cuadernos de Jupyter proporcionados:

1.  Inicia Jupyter Lab o Notebook: `jupyter notebook`.
2.  Abre el archivo `analisis_sismico.ipynb` (o el nombre correspondiente).
3.  Ejecuta las celdas para visualizar los mapas y estadísticas.

## 📚 Fuentes de Datos

Los datos utilizados en este proyecto provienen de fuentes oficiales como:

  - [IG-EPN](https://www.igepn.edu.ec/) (Instituto Geofísico de la Escuela Politécnica Nacional - Ecuador).
  - [USGS](https://earthquake.usgs.gov/) (United States Geological Survey).
